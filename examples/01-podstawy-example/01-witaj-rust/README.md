## Opis

Podstawowy przykład "Hello World" w Rust, pokazujący:
- Strukturę projektu Cargo
- Funkcję `main()`
- Makro `println!`

---

## Uruchomienie

```bash
cargo run
```

## Sprawdzenie

```bash
cargo check
```

## Formatowanie

```bash
cargo fmt
```

---

## Struktura Projektu

```
01-witaj-rust/
├── Cargo.toml    # Konfiguracja projektu
├── src/
│   └── main.rs   # Główny plik źródłowy
└── target/       # Build artifacts (nie commitowane)
```
